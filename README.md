## 📋 Overview
This repository is a **forked workspace** used for my Low-Level Design (LLD) coursework. I use this space to solve design problems, refactor legacy code, and implement robust software architectures.

The primary goal is to transform provided "starter code" into clean, maintainable systems that adhere to industry-standard design principles.

---

## 🛠️ Assignment Workflow
Since this is a forked repository, my contribution involves:
1. **Analysis:** Identifying design smells and anti-patterns in the provided assignment base.
2. **Refactoring:** Applying **SOLID Principles** to decouple logic and improve extensibility.
3. **Implementation:** Adding new design components (e.g., `SimplePrinter.java`) to meet project requirements.
4. **Consistency:** Daily commits to document the evolution of my design logic.

## 📂 Progress Tracker

### Phase 1: SOLID Principles 🧩
Refactoring Exercises **Ex1 to Ex10** to master the five core pillars:
* **SRP (Single Responsibility):** Modularizing classes for better maintainability.
* **OCP (Open-Closed):** Designing for extension without modification.
* **ISP (Interface Segregation):** Featured in my recent updates to `SimplePrinter.java`.

Implementing behavioral and structural patterns to solve complex architectural challenges.
